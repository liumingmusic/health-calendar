# 养生日历 · 天人合一 顺应四时

一个**纯静态、零构建、零 API、完全离线**的中国传统时令养生小站。

以「天人合一、顺应四时」为总纲，主线为 **二十四节气**，辅以 **七十二候**（物候变化）与 **十二时辰**（子午流注、当令经络脏腑）养生。立足中医时令观：春养肝、夏养心、长夏养脾、秋养肺、冬养肾；配导引术（八段锦 / 五禽戏）、药食同源食材，以及道家「守静」「吐纳」「抱一」「坐忘」等顺时养生法。道家智慧点到即止，不涉宗教仪轨。

## 本地预览

```bash
# 1. （可选）重新生成节气日期表
npm run gen

# 2. 启动本地静态服务器（fetch 需经 http，勿用 file:// 直接打开）
npm run serve
# 浏览器访问 http://localhost:8080
```

## 数据来源（全部离线，免 key）

- `solar-terms.json`：由 `scripts/gen-terms.js` 用 Meeus 天文算法计算太阳视黄经，二分查找每个节气穿越目标经度的时刻，折算为北京时间。覆盖 **2024–2035**，2026 年各节气日期已核对准确。
- `health.json`：24 节气各自的时令养生（脏腑、气候、饮食、起居、导引、穴位、宜忌），逐条编写，体现季节差异。
- `hou.json`：七十二候，按节气分组，每节气 3 候（候名 + 物候白话解释）。
- `hours.json`：十二时辰（子午流注），含时段、当令经络脏腑、养生宜忌、经络配色。
- `daoism.json`：道家养生数据库，含《道德经》《庄子》原句（16 条）、正经养生功法（守静、坐忘、抱一、六字诀、踵息、导引、内丹凝神气穴等 7 项）、四时道家顺时养生要旨（5 条）。

## 功能

- 头部：今日公历日期 + 当前节气大标识（距下一节气 N 天）+ 当前时辰标识。
- 今日养生主卡：气候、当令脏腑、养生总纲；饮食（药食同源）、起居宜忌、导引术、保健穴位；当前时辰（子午流注）宜忌；顶部「道经今读」轮播原句。
- 道家养生 Tab：道经原句典藏、7 项正经功法（含原句/步骤/功效/注意）、四时道家要旨，当前季节高亮。
- 七十二候区：当前节气三候，可展开查看全年候历。
- 打卡区：喝够水 / 早睡 / 八段锦 / 食养 微习惯，存 localStorage，显示连续打卡天数。
- 全年总览：24 节气网格，季节配色，当前节气呼吸感高亮；点击任一点开详情；季节筛选 Tab。
- 页脚每日道家养生格言（随日期轮播）。

## 部署到 GitHub Pages

本仓库为纯静态站点，直接推送 `main` 分支根目录，于仓库 Settings → Pages 选择 **Deploy from a branch / main / root** 即可。
