# 养生日历 · 天人合一 顺应四时

一个**纯静态、零构建、零 API、完全离线**的中国传统时令养生小站。

以「天人合一、顺应四时」为总纲，主线为 **二十四节气**，辅以 **七十二候**（物候变化）、**十二时辰**（子午流注、当令经络脏腑）养生，并融入**奇门遁甲、黄历、八字、天干地支、方位、经络取穴、每日签运、生肖每日运程**等传统历法术数与文化。另设「使用说明」Tab，用大白话解释各板块与术语，帮助用户快速看懂。立足中医时令观：春养肝、夏养心、长夏养脾、秋养肺、冬养肾；配导引术（八段锦 / 五禽戏）、药食同源食材，以及道家「守静」「吐纳」「抱一」「坐忘」等顺时养生法。道家智慧、术数与文化参考均点到即止，不涉宗教仪轨，不替代专业命理或医疗建议。

## 本地预览

```bash

# 2. 启动本地静态服务器（fetch 需经 http，勿用 file:// 直接打开）
npm run serve
# 浏览器访问 http://localhost:8080
```

## 数据来源（全部离线，免 key）

- `solar-terms.json`：由 `scripts/gen-terms.js` 用 Meeus 天文算法计算太阳视黄经，二分查找每个节气穿越目标经度的时刻，折算为北京时间。覆盖 **2024–2035**，2026 年各节气日期已核对准确。
- `health.json`：24 节气各自的时令养生（脏腑、气候、饮食、起居、导引、穴位、宜忌），逐条编写，体现季节差异。
- `hou.json`：七十二候，按节气分组，每节气 3 候（候名 + 物候白话解释）。
- `hours.json`：十二时辰（子午流注），含时段、当令经络脏腑、养生宜忌、经络配色，以及十二正经的**原穴/络穴/募穴**。
- `daoism.json`：道家养生数据库，含《道德经》《庄子》原句（16 条）、正经养生功法（守静、坐忘、抱一、六字诀、踵息、导引、内丹凝神气穴等 7 项）、四时道家顺时养生要旨（5 条）。
- `fortune-sticks.json`：每日一签数据库，30 支修身养性签，含签级（上上/上/中/下/凶）、签诗（四句）、解曰、指、宜忌。

## 功能

- 头部：今日公历日期 + 古历罗盘时钟（二十四节气环、八卦环、八门环、方位环、十二时辰环 + 实时时分秒指针）+ 当前节气 / 当前时辰双焦点大字。
- **今日总览**：首页首屏汇总卡，一屏看清今日公历/农历干支、当前节气+季节、当前时辰+当令脏腑、八字四柱、今日宜忌、道家功课、今日一签、生肖运程。
- 今日养生主卡：气候、当令脏腑、养生总纲；饮食（药食同源）、起居宜忌、导引术、保健穴位；**本时辰经络取穴（原穴/络穴/募穴）**；当前时辰（子午流注）宜忌；顶部「道经今读」轮播原句。
- 黄历八字 Tab：顶部「**今日一眼通**」白话摘要卡；下方四柱、奇门遁甲、黄历宜忌、天干地支 4 个可折叠区块，默认收起较难的奇门遁甲与天干地支，降低信息冲击。其中**奇门遁甲**九宫洛书排盘（阴阳遁、局数、旬首、值符/值使、九星/八门/八神/三奇六仪，带辉光动画）为文化示意；四柱命盘（年/月/日/时 干支、五行、纳音）、五行统计、日主简析；**天干地支全览**（十干、十二支、六十甲子，今日干支高亮）；黄历宜忌（建除、值神黄黑道、冲煞、彭祖百忌）；喜神/财神/福神/太岁/岁破/三煞方位；当前时辰经络取穴。
- 道家养生 Tab：**「今日道家功课」按日轮换**（主修功法 + 诵持道经 + 今日心法 + 当令要旨），每日依公历日期推算，**每天不同、人人相同**；功法全录、道经全览改为**折叠手风琴**，默认显示提要，点击展开详情；当前季节高亮。
- 每日签运 Tab：**每日一签**（30 支修身养性签，按日固定，可「再摇一签」娱乐），签筒为**写实竹签筒图片**（竹纹圆柱 + 红箍 + 八卦太极贴 + 插签），摇签时整筒摆动 5 秒、竹签跳出覆盖筒身、签文揭示动画、解曰/指/宜忌、解签提示；**今日运程**（十二生肖按日支六合/三合/刑冲害等关系评分排定每日运程，本日干支综论、建除与值神解读）。
- 使用说明 Tab：用大白话讲解「今日养生 / 黄历八字 / 道家养生 / 每日签运 / 全年总览」怎么看，并附术语小词典（节气、候、时辰、干支、八字、纳音、奇门遁甲、建除、值神等），顶部有本日速览（今日节气、当前时辰、今日日干支）。
- 响应式：桌面与手机自适应；手机端 Tab 可横向滑动，宽表格可横向滚动，确保无溢出。
- 七十二候区：当前节气三候，可展开查看全年候历。
- 打卡区：喝够水 / 早睡 / 八段锦 / 食养 微习惯，存 localStorage，显示连续打卡天数。
- 全年总览：24 节气网格，季节配色，当前节气呼吸感高亮；点击任一点开详情；季节筛选 Tab。
- 页脚每日道家养生格言（随日期轮播）。

## 部署到 GitHub Pages

本仓库为纯静态站点，直接推送 `main` 分支根目录，于仓库 Settings → Pages 选择 **Deploy from a branch / main / root** 即可。
